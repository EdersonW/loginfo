# TESTE DE APTIDÃO 

Para instalar a aplicação, siga as instruções abaixo:

-- Instalar as dependências:
```bash
composer install
```

-- Criar banco de dados:</br>
Nome: loginfo </br>
Charset: UTF8 - utf8_unicode_ci</br>
Database: MySQL</br>

-- Iniciar a migração utilizando Phinx
```bash
vendor/bin/phinx migrate
```
